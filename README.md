**ScrollSound**是一个可以让你在任务栏空白处用滚轮控制音量的小工具。一个看似鸡肋用了却离不开的小功能。<br>
目前支持win10、win11。<br>

## ✨使用

双击打开，将鼠标移至任务栏空白处，滚动滚轮或按下滚轮即可调节音量大小或静音。

为什么要用？为了优雅。
